****************此处一定注意了*****************
**********************************************
很多朋友再安装之后，会出现找不到路径的问题，这里的
主要原因是我使用URL_MODEL为rewrite模式，有两种方法
可解决这个问题：
1.修改HOME中的config.php配置，将URL_MODEL改为1.
2.在apache的httpd.conf中加载mod_rewrite模块，并将AllowOverride None 将None改为 All。
